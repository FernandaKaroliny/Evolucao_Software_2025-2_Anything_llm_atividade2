A estratégia de branch utilizada no projeto AnythingLLM é o "git flow". O git flow é um modelo de gerenciamento de branches que foi projetado para facilitar o desenvolvimento de software com uma equipe. Ele define um conjunto de branches principais e secundárias que são usados para diferentes fases do ciclo de vida de um projeto, como desenvolvimento, teste e lançamento.
